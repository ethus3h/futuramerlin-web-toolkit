Command usage: crystallise

Page location: Software Crystallise

Overview

This command is part of the Crystallise package; see the package page for installation instructions.

This command provides the main functionality of Crystallise, an app to easily archive files stored in computer-accessible digital formats.

Documentation

Usage

Syntax

If relative path names are used for <files>, the file(s) being crystallised must be within the current directory. The script tests for some simple violations of this requirement, but not all of them.
