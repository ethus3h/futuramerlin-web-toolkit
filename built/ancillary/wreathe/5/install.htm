Installation instructions for Wreathe 5: Rain

Page location: Software Wreathe Wreathe 5: Rain

Table of Contents

  1. System requirements
  2. Installation

System requirements

  • Intel Pentium 4 CPU or compatible, 2 GHz or faster, hyperthreaded or with two cores
  • 30GB available disk space recommended
  • Compatible GPU (known to work well: ATI Radeon 9600 with 128 MB video memory, Intel 945GM with 256 MB video memory)
  • Colour display, LCD (1280x1024 or 1024x768) recommended
  • At least 720MB RAM, 2 to 3 GB recommended
  • Compatible keyboard (any standard IBM-compatible should work, such as the IBM SpaceSaver II)
  • Compatible pointing tool (TrackPoint, mouse, etc.)
  • DVD burner

It is probably possible to install Wreathe 7.3 on computers that do not meet the recommended system requirements, but that is not supported.

Installation

Wreathe 5: Rain is distributed as a DVD image. To install it:

  • Burn the disk image to a DVD. (Use slow burning speeds to prevent data corruption, and verify the burned DVD afterwards.)
  • Boot your computer from the DVD. The username is kyan and the password is elen-esar. The system uses the Dvorak keyboard layout by default.
  • Press [F9], type gksu ubiquity gtk_ui, and press [enter].
  • Follow the instructions on screen. WARNING: Formatting a partition or drive will erase ALL the data on the partition or drive being formatted. Having a complete and verified backup of all your data is strongly recommended.
  • Restart the computer without the DVD after the installation is complete.
